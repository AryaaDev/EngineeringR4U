Metode Eliminasi Gauss

Contoh soal SPL tiga variabel Metode Eliminasi Gauss

Cara Kerja Kalkulator Metode Eliminasi Gauss

  1. Menyusun SPL ke dalam matriks augmented [A|B]
  2. Memilih pivot terbesar pada setiap kolom (Partial Pivoting)
  3. Melakukan OBE hingga terbentuk matriks segitiga atas
  4. Menentukan jenis solusi berdasarkan rank matriks
  5. Menghitung solusi dengan substitusi balik (Back Substitution)

Contoh Inputan Eliminasi Gauss

Masukkan jumlah variabel, klik Buat Matriks, lalu isi nilai matriks sesuai soal.

Format Input 
         Kolom 1| Kolom 2 | Kolom 3 | Konstanta
Baris 1 :   2   |    4    |   -2    |     2
Baris 2 :   4   |    9    |   -3    |     8
Baris 3 :  -2   |   -3    |    7    |    10

Metode Gauss Seidel

Contoh SPL tiga variabel menggunakan metode Gauss-Seidel dengan solusi awal (1, 2, 2) dan toleransi 0,3.

Cara Kerja Kalkulator Metode Gauss-Seidel

  1. Mengubah SPL ke bentuk xᵢ = (bᵢ − Σaᵢⱼxⱼ) / aᵢᵢ
  2. Menghitung x₁, x₂, ..., xₙ secara berurutan dari tebakan awal
  3. Menggunakan nilai terbaru x₁, x₂, ... yang telah diperoleh pada iterasi yang sama
  4. Menghitung galat relatif Ex = |(xbaru − xlama) / xbaru|
  5. Mengulangi iterasi hingga Ex < ε atau iterasi maksimum tercapai

Contoh Inputan Gauss-Seidel

Tentukan jumlah variabel, klik Buat Matriks, lalu isi nilai matriks. Solusi awal, toleransi (ε), dan iterasi bersifat opsional.

Format Input 
         Kolom 1 | Kolom 2 | Kolom 3 | Konstanta
Baris 1 :   4    |   -1    |    1    |     7
Baris 2 :   4    |   -8    |    1    |   -21
Baris 3 :  -2    |    1    |    5    |    15
------------------------------------------------
Solusi Awal : 1 | 2 | 2 |
Toleransi   : 0.03
Iterasi     : 50

Catatan:
Toleransi(ε) (Opsional) → Jika kosong menggunakan nilai default 0.0001
Maksimum Iterasi (Opsional) → Jika kosong menggunakan nilai default 50
Solusi Awal (Opsional) → Jika kosong menggunakan nilai awal (0,0,...,0)

Metode Bisection

Contoh pencarian akar persamaan menggunakan metode Bisection dengan interval awal a = 2 dan b = 3, ketelitian 3 digit, serta toleransi 0.0001.

Cara Kerja Kalkulator Metode Bisection

  1. Masukkan fungsi f(x), batas bawah (a), batas atas (b), dan toleransi (ε). Pastikan f(a) dan f(b) memiliki tanda berbeda agar akar berada di dalam selang.
  2. Hitung titik tengah selang:
    c = (a + b) / 2 lalu evaluasi nilai f(a), f(c), dan f(b).
  3. Tentukan selang baru berdasarkan perubahan tanda:
    • f(a) × f(c) < 0 → b = c
    • f(a) × f(c) > 0 → a = c
  4. Hitung estimasi galat |b − c| dan ulangi proses hingga galat memenuhi toleransi atau akar eksak ditemukan.
  5. Nilai c terakhir digunakan sebagai hampiran akar persamaan.

Contoh Inputan Bisection

Isi fungsi f(x), interval a dan b, ketelitian, serta toleransi.

Format Input 
Fungsi f(x) : x^3-4*x-9
a           : 2
b           : 3
Ketelitian  : 3
Toleransi   : 0.0001

Catatan:
- f(a) dan f(b) harus memiliki tanda berbeda
- Ketelitian (opsional): jumlah digit desimal hasil, default 3 digit
- Toleransi (opsional): batas galat iterasi, default 0.0001

Metode Secant

Contoh pencarian akar persamaan menggunakan metode Secant dengan tebakan awal x₀ = 2 dan x₁ = 3, serta maksimum 10 iterasi.

Cara Kerja Kalkulator Metode Secant

  1. Masukkan fungsi f(x), tebakan awal x₀ dan x₁, serta jumlah iterasi maksimum(opsional)
  2. Hitung nilai fungsi pada kedua tebakan:
    f(x₀) dan f(x₁).
  3. Hitung pendekatan akar berikutnya menggunakan persamaan secant:
    x₂ = x₁ − f(x₁)(x₁ − x₀)/(f(x₁) − f(x₀)).
  4. Hitung galat |x₂ − x₁|, kemudian geser nilai:
    x₀ = x₁ dan x₁ = x₂.
  5. Ulangi hingga galat sangat kecil atau batas iterasi tercapai. Nilai terakhir menjadi hampiran akar.

Contoh Inputan Secant

Isi fungsi f(x), nilai awal x₀ dan x₁, serta jumlah iterasi.

Format Input 
Fungsi f(x) : x^3-4*x-9
x0          : 2
x1          : 3
Iterasi     : 10

Catatan:
- x0 dan x1 merupakan dua tebakan awal akar
- Iterasi menentukan jumlah maksimum perulangan(Opsional)

Metode Interpolasi Newton

Contoh interpolasi Newton menggunakan data yang diberikan untuk memperkirakan nilai fungsi pada x = 2.5 dan menghitung galat interpolasinya.

Cara Kerja Kalkulator Metode Interpolasi Newton

  1. Masukkan titik data x dan y serta nilai target x yang akan dicari.
  2. Susun tabel beda terbagi untuk memperoleh koefisien b₀, b₁, b₂, ..., bₙ.
  3. Bentuk polinom interpolasi Newton:
    P(x) = b₀ + b₁(x−x₀) + b₂(x−x₀)(x−x₁) + ...
  4. Substitusikan nilai target x ke dalam polinom yang terbentuk.
  5. Hasil perhitungan memberikan nilai hampiran fungsi pada titik yang dicari.

Contoh Inputan Interpolasi Newton

Isi data x, data y, dan nilai x yang dicari.

Format Input 
Data x      : 1 2 3 4
Data y/f(x) : 2 5 10 17
Nilai x     : 2.5
Fungsi Asli : x^2+1

Catatan:
- Data x dan data y harus berjumlah sama
- Pisahkan setiap data menggunakan spasi
- Fungsi Asli bersifat opsional untuk menghitung galat

Metode Integrasi Trapesium

Contoh perhitungan integral menggunakan metode Trapesium pada interval [0, 2] dengan 4 subinterval.

Cara Kerja Kalkulator Metode Integrasi Trapesium

  1. Masukkan fungsi f(x), batas integral a dan b, serta jumlah pias (n).
  2. Hitung lebar interval:
    h = (b − a)/n.
  3. Tentukan titik partisi x₀, x₁, ..., xₙ dan hitung nilai f(x) pada setiap titik.
  4. Terapkan rumus Trapesium:
    I = (h/2)[f(x₀)+2Σf(xᵢ)+f(xₙ)].
  5. Hasil perhitungan digunakan sebagai nilai hampiran integral pada interval yang diberikan.

Contoh Inputan Integrasi Trapesium

Isi fungsi f(x), batas a dan b, serta jumlah subinterval.

Format Input 
Fungsi f(x) : x^2
a           : 0
b           : 2
n           : 4

Catatan:
- n adalah jumlah subinterval (pias)
- Semakin besar n, hasil biasanya semakin akurat

Metode Runge Kutta

Contoh penyelesaian persamaan diferensial menggunakan metode Runge-Kutta Orde 4 dengan kondisi awal x₀ = 0, y₀ = 2, langkah h = 0.1, dan 5 iterasi.

Cara Kerja Kalkulator Metode Runge-Kutta Orde 4

  1. Masukkan persamaan diferensial dy/dx = f(x,y), nilai awal (x₀,y₀), langkah h, dan jumlah iterasi.
  2. Hitung empat kemiringan:
    • k₁ = h·f(xₙ,yₙ)
    • k₂ = h·f(xₙ+h/2, yₙ+k₁/2)
    • k₃ = h·f(xₙ+h/2, yₙ+k₂/2)
    • k₄ = h·f(xₙ+h, yₙ+k₃)
  3. Hitung perubahan nilai y:
    Δy = (k₁ + 2k₂ + 2k₃ + k₄)/6.
  4. Perbarui nilai:
    xₙ₊₁ = xₙ + h
    yₙ₊₁ = yₙ + Δy.
  5. Ulangi hingga jumlah langkah selesai. Nilai terakhir menjadi solusi hampiran persamaan diferensial.

Contoh Inputan Runge-Kutta Orde 4

Isi fungsi dy/dx = f(x,y), nilai awal x₀ dan y₀, langkah h, serta iterasi.

Format Input 
Fungsi dy/dx : x+y
x0           : 0
y0           : 2
h            : 0.1
Iterasi      : 5

Catatan:
- Fungsi dapat menggunakan variabel x dan y
- h adalah ukuran langkah (step size)
- Iterasi menentukan banyaknya langkah perhitungan RK4
